# Kabal Solar Company Website

This is the official website source code for **Kabal Solar Company**, a trusted provider of solar panel installations and electric geyser solutions in Swat Valley, Pakistan.

## Project Overview

The website serves as a digital storefront for the business, allowing customers to:
- View available services (Solar Systems, Electric Geysers).
- Request free quotes via an online form.
- See project portfolios and customer testimonials.
- Contact the company directly via Phone, WhatsApp, or Email.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Service Showcase**: Detailed sections for Solar Panels and Electric Geysers.
- **Quote System**: Functional PHP-backend form (`submit_quote.php`) for collecting customer inquiries.
- **Admin View**: Restricted area (`view_quotes.php`) to view submitted quotes.
- **Interactive Elements**: Smooth scrolling, sticky header, and mobile navigation.

## Technology Stack

- **Frontend**: HTML5, CSS3 (Custom Grid/Flexbox), JavaScript (Vanilla).
- **Backend**: PHP (for form processing and data handling).
- **Icons**: Font Awesome 6.
- **Fonts**: Google Fonts (Inter & Poppins).

## Setup & Deployment

1.  **Server Requirements**: A web server with PHP support (e.g., Apache/Nginx with PHP 7.4+).
2.  **Local Development**:
    - Install XAMPP, WAMP, or MAMP.
    - Place the project folder in `htdocs` or `www`.
    - Start the Apache server.
    - Visit `http://localhost/Solar-Web-project/` in your browser.
3.  **Configuration**:
    - Check `config.php` for database or email configuration settings (if applicable).
