## Image Optimization Tips

1. **Compress images** before uploading to reduce file size
   - Use: TinyPNG.com or Squoosh.app
   - Target: Under 200KB for photos, under 50KB for logos

2. **Use appropriate formats**:
   - Logo/badges: PNG (supports transparency)
   - Photos: JPG (smaller file size)
   - Modern browsers: Consider WebP format

3. **Naming convention**:
   - Use lowercase
   - Use hyphens instead of spaces
   - Be descriptive: `solar-installation.jpg` not `img1.jpg`

4. **Resolution**:
   - Use 2x resolution for retina displays
   - Example: For 400px display, use 800px image

## How to Add Images
